💰 Personal Finance Management System

A secure backend REST API built using Spring Boot for managing personal finances such as income, expenses, goals, and monthly spending limits.

This project demonstrates modern backend development practices including JWT authentication, RESTful APIs, DTO architecture, and MySQL database integration.

🚀 Features
✅ User Authentication & Authorization (JWT)
✅ Add / Update / Delete Income & Expenses
✅ Category-wise Expense Tracking
✅ Monthly Spending Threshold Limit
✅ Financial Goal Management
✅ Notes & Tags for transactions
✅ RESTful API Architecture
✅ OpenAPI / Swagger Documentation

🛠️ Tech Stack
Technology	Usage
☕ Java 17	Core Programming
🌱 Spring Boot	Backend Framework
🔐 JWT	Authentication & Security
🗄 MySQL	Database
📦 Maven	Dependency Management
📑 OpenAPI / Swagger	API Documentation
🐳 Docker	Containerization
